# yeboahs-law-visual-abstract-2026
Visual Abstract for "Yeboah's Law: A Dimensionless Threshold Criterion for Dynamic Impact Cutting" submitted to Journal of Manufacturing Processes 2026. Demonstrates η ≥ 1 threshold and Yeboah Constant k_Y across waterjet, chisel, and razor cutting. From Technical SHS for the Deaf, Mampong-Akuapem, Ghana.
